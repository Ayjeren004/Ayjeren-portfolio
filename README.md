# Premium 3D Bento Box Portfolio & Interactive Labs

A responsive, high-performance Vite + React developer portfolio designed as a premium grid-based dashboard. It showcases professional software engineering history, academic qualifications, and built-in interactive simulator consoles.

## 🚀 Live Demo
Visit the live deployment at: **[https://ayjeren-portfolio.web.app](https://ayjeren-portfolio.web.app)**

## 🧪 Interactive Showcase Features
*   **3CX CRM Call Pop-up Simulator:** Models an incoming call telephony connector. Includes CRM detail lookup fields ( Alexis Daniel ), call state events, Call Wrap-up dropdown routing, Odoo AI text refinement simulation, follow-up scheduling, and simulated chatter logs.
*   **Precipitation GPS Attenuation Study:** A dynamic HTML5 canvas mapping satellite signal dampening curves mathematically based on selectable precipitation modes (Clear Sky, Rain, Thunderstorm).
*   **Niko OS Terminal:** A mock command-line interface validating custom interactive prompts.

## 🛠️ Tech Stack
*   **Frontend Core:** React, Vite (Fast HMR)
*   **Styling:** Vanilla CSS & Tailwind CSS utility configurations
*   **Icons:** Lucide React icons
*   **Canvas Graphics:** HTML5 Canvas API (custom rendering loops)

## 💻 How to Run Locally

To clone, install, and run this project on your local machine, follow these steps:

### 1. Clone the Repository
```bash
git clone https://github.com/Ayjeren004/Ayjeren-portfolio.git
cd Ayjeren-portfolio
```

### 2. Install Dependencies
Ensure you have [Node.js](https://nodejs.org) installed, then run:
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```
Once started, open your browser and navigate to the local address (typically **`http://localhost:5173`**).

### 4. Build for Production
To generate a compiled production bundle inside the `dist/` directory:
```bash
npm run build
```
